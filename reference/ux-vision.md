# Experiencia AtemiMX

Este documento captura la experiencia ideal que debe vivir el docente al usar AtemiMX; una sola herramienta que combina la gestión escolar, la evaluación formativa, el alivio burocrático y el reconocimiento significativo con un tono pedagógico, intuitivo y seguro.

## 1. Gestión de aula eficiente y flexible

### Cuaderno digital
- El docente percibe el módulo como el “cuaderno definitivo”, con columnas ilimitadas, editores rápidos y soporte para múltiples tipos de notas: numéricas, texto, íconos o combinaciones que reflejen la escala oficial y la naturaleza cualitativa de la NEM.
- Cada celda puede anexar recursos multimedia (foto, audio, video) para documentar evidencia directa del progreso estudiantil.
- El foco está en mantener todos los datos visibles en una sola pantalla y permitir acciones rápidas para captura y edición.

### Organización visual
- Los datos se agrupan con pestañas configurables que permiten cambiar el foco por grupo, materia o ciclo evaluativo sin perder contexto.
- Los filtros y agrupaciones ayudan a mantener la pantalla ordenada pero accesible, como una interfaz “modular como una navaja suiza”.

### Plano de asientos
- El seating plan funciona como herramienta principal de observación: asiste en asistencia, anotación rápida de incidencias y registro de notas desde la vista del aula.
- El docente puede alternar entre el plano y el cuaderno guardando trazabilidad directa entre la ubicación del estudiante y su evaluación.

## 2. Evaluación formativa y centrada en el proceso

### Diagnóstico inicial y adaptación curricular
- La experiencia guía al docente para diseñar, aplicar y analizar evaluaciones diagnósticas (KPSI, redes sistémicas u otras) para identificar prerrequisitos.
- Los resultados se usan automáticamente para ajustar la programación escolar, buscando alineación entre el programa y la capacidad cognitiva de cada estudiante.

### Retroalimentación continua
- La interfaz pregunta y documenta tres momentos clave: “¿Cuál es la tarea?”, “¿Cómo lo estoy haciendo?” y “¿Cuál es el próximo paso?”, con soporte para comentarios multimedia.
- Se habilita la autoevaluación y la coevaluación mediante preguntas detonadoras que hacen al estudiante responsable de su proceso.

### Rúbricas
- Crear rúbricas personalizadas es ágil: se pueden definir criterios, niveles de desempeño y mensajes guía; todo enlazado directamente con las tareas del cuaderno digital.

### Trazabilidad curricular (LOMLOE/NEM)
- Cada columna o rúbrica se conecta con criterios, competencias específicas, descriptores operativos y saberes básicos, respetando la jerarquía curricular.
- Esta integración permite ver de inmediato qué elementos curriculares se están evaluando en cada actividad.

### Visualización de logro
- Widgets visuales muestran el progreso por criterios, saberes, competencias y el Perfil de Salida: pueden ser numéricos, círculos de color o texto narrativo.
- El “GPS de aprendizaje” ofrece mensajes de avance y referencias para planes de mejora.

## 3. Alivio de la carga administrativa y burocrática

### Codiseño curricular colaborativo
- Se capturan y editan documentos, matrices y rutas desde espacios compartidos para que los colectivos docentes construyan y validen su Programa Analítico de forma autónoma.

### Gestión de infraestructura (RM-08)
- Un submódulo facilita solicitudes de mantenimiento o siniestro, capturando evidencia fotográfica, croquis y justificaciones de riesgo para la Dirección Operativa.
- Las solicitudes se pueden asociar con lugares físicos y responsables para acelerar respuestas institucionales.

### Autonomía profesional
- El sistema no fuerza formatos reglamentarios: el docente decide qué evidencia comparte y mantiene control sobre sus procesos pedagógicos.
- La plataforma se percibe como un sistema de sistematización, no como una herramienta de vigilancia administrativa.

### Seguridad
- El AppShell exige correo institucional y contraseña fuerte para el login, verificando la plantilla docente autorizada antes de habilitar cualquier captura.
- El acceso respeta roles (docente, dirección, prefectura, etc.) y solamente desbloquea funciones después de validaciones previas.

## 4. Reconocimiento y motivación

### Insignias digitales
- Con herramientas parecidas a “Badge Maker” el docente crea insignias para habilidades blandas, competencias adquiridas y elementos transversales.
- Las insignias tienen puntajes que pueden incorporarse al cuaderno mediante la fórmula `BADGEMAKER()`, permitiendo que el reconocimiento impacte en los indicadores de logro.
- El sistema promueve celebraciones de avance y visibilidad de logros más allá de calificaciones numéricas.

## Conclusión

Al cerrar la sesión, el docente debe sentir que AtemiMX fue su aliado pedagógico y administrativo: una plataforma segura, flexible y motivadora que lo libera de tareas farragosas para concentrarse en acompañar procesos de aprendizaje profundos y competenciales.
